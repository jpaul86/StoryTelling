# README
Datasets ready to be used for the DataViz Portfolio Project:

## listings.csv: 
- Airbnb listings from Berlin from 2021-09-21 
- Source: http://insideairbnb.com/get-the-data.html
- Additional data available on above website

## weather_timeseries.csv
- Long timeseries of weather data from selected German sites
- Source: Deutscher Wetterdienst DWD, obtained via R package rdwd
- The data is annual data; some indicators may be difficult to interpret at first glance
- For precise variable definitions, see the file wether_metadata.csv

## airquality-2020.csv and airquality-timeseries.csv
- Cross-section of annual data for all measurement sites in Europe in 2020 (values represent annual averages)
- Time series of daily values for selected German sites since 2004 (values represent daily averages)
- Source: European Environmental Agency, obtained via R package saqgetr
- Basic data cleaning has already been carried out due to the large number of missing values
